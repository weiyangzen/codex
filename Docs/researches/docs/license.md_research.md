# license.md 研究文档

## 场景与职责

license.md 是 Codex CLI 项目的许可证说明文档。该文档非常简洁，仅说明项目使用 Apache-2.0 许可证。

**适用场景：**
- 用户需要了解项目的许可证信息
- 开发者确认使用/贡献的法律条款
- 合规审查

## 功能点目的

### 1. 许可证声明
- **许可证**：Apache License 2.0
- **位置**：`../LICENSE`（项目根目录）
- **目的**：明确项目的开源许可证

### 2. 合规指引
- 指向完整的许可证文件
- 确保用户了解使用条款

### Apache-2.0 许可证要点

1. **授权**
   - 授予专利许可
   - 允许商业使用
   - 允许修改和分发

2. **要求**
   - 保留版权声明
   - 保留免责声明
   - 变更说明

3. **限制**
   - 不提供担保
   - 责任限制

## 风险、边界与改进建议

### 潜在风险

1. **许可证兼容性**
   - 与其他开源许可证的兼容性
   - 建议：定期审查依赖项许可证

2. **版权声明完整性**
   - 确保所有源文件包含适当的版权声明
   - 建议：添加许可证头检查工具

### 边界情况

1. **双重许可**
   - 某些组件可能有不同的许可证
   - 需要明确标识

2. **专利授权**
   - Apache-2.0 包含专利授权条款
   - 需要确保合规

### 改进建议

1. **许可证头**
   - 为所有源文件添加许可证头
   - 使用自动化工具检查

2. **SBOM**
   - 生成软件物料清单
   - 列出所有依赖和许可证

3. **许可证扫描**
   - 集成 CI 许可证扫描
   - 自动检测许可证冲突

4. **贡献者协议**
   - 与 CLA 协调
   - 确保贡献者了解许可证条款

5. **文档增强**
   - 添加许可证摘要
   - 提供常见问题解答
