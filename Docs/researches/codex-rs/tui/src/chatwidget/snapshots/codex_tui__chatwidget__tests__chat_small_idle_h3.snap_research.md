# 研究文档: chat_small_idle_h3.snap

## 场景与职责

该快照文件测试 ChatWidget 在小屏幕（高度为3行）空闲状态下的渲染效果。验证在有限但可用的显示空间下UI的正确表现。

## 功能点目的

1. **最小可用UI**: 确保在3行高度下UI仍然可用
2. **关键信息显示**: 验证最重要的信息能够显示
3. **空闲状态优化**: 测试空闲状态下的简化显示

## 具体技术实现

### 渲染内容（3行高度）

在3行高度下，UI可能显示：
- 状态栏
- 简短提示信息
- 输入区域或操作提示
