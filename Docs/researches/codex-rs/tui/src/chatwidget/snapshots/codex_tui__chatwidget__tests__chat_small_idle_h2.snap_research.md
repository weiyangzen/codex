# 研究文档: chat_small_idle_h2.snap

## 场景与职责

该快照文件测试 ChatWidget 在小屏幕（高度为2行）空闲状态下的渲染效果。验证在紧凑显示空间下UI的正确表现。

## 功能点目的

1. **小屏幕适配**: 确保UI在2行高度时能显示关键信息
2. **空闲状态UI**: 验证无活动时的简化UI状态
3. **渐进式显示**: 测试比1行高度稍好的显示效果

## 具体技术实现

### 渲染内容（2行高度）

在2行高度下，UI可能显示：
- 第1行：状态指示器或标题
- 第2行：输入提示或快捷操作

## 关键代码路径与文件引用

- **测试文件**: `codex-rs/tui/src/chatwidget/tests.rs`
- **布局逻辑**: 高度分配算法
