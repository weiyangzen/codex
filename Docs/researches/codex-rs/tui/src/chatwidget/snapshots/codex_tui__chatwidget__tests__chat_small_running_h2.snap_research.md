# 研究文档: chat_small_running_h2.snap

## 场景与职责

该快照文件测试 ChatWidget 在小屏幕（高度为2行）运行状态下的渲染效果。

## 功能点目的

1. **紧凑运行UI**: 在2行高度下显示任务运行状态
2. **状态+操作**: 尝试同时显示状态和可能的操作
3. **渐进式信息**: 比1行高度稍好的信息显示

## 具体技术实现

### 渲染内容

2行高度运行状态可能显示：
- 第1行：活动指示器 + 简短状态
- 第2行：取消提示或进度信息
