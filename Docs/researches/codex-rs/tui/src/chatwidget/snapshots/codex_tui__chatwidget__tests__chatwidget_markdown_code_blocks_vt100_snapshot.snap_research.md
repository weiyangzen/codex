# 研究文档: chatwidget_markdown_code_blocks_vt100_snapshot.snap

## 场景与职责

该快照文件测试 ChatWidget 在 VT100 终端下渲染 Markdown 代码块的效果。验证代码块在传统终端环境下的正确显示。

## 功能点目的

1. **代码块渲染**: 在VT100终端下正确显示Markdown代码块
2. **语法高亮降级**: 在有限颜色支持下的语法高亮适配
3. **格式保持**: 保持代码格式和缩进

## 具体技术实现

### Markdown代码块处理

```rust
// 代码块渲染使用简化的高亮方案
// VT100仅支持8色，需要降级复杂的语法高亮
```

### 渲染特点

- 代码块边框使用ASCII字符
- 语法高亮使用基本颜色
- 保持代码缩进和换行

## 关键代码路径与文件引用

- **测试文件**: `codex-rs/tui/src/chatwidget/tests.rs`
- **Markdown渲染**: `codex-rs/tui/src/markdown.rs`
- **语法高亮**: 使用简化的高亮规则

## 依赖与外部交互

1. **syntect** 或类似库: 语法高亮（降级模式）
2. **pulldown-cmark**: Markdown解析

## 风险、边界与改进建议

### 风险
- 复杂的语法高亮在VT100下可能无法呈现
- 代码块可能超出屏幕宽度

### 改进建议
1. 为VT100提供专门的代码块样式
2. 添加代码块水平滚动
3. 考虑代码折叠功能
