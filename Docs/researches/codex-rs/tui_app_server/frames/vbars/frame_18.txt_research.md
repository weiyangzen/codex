# frame_18.txt 研究文档

## 场景与职责

`frame_18.txt` 是 Codex TUI 应用服务器欢迎界面动画的 ASCII 艺术帧文件，属于 **vbars**（垂直条形图）动画变体的第 18 帧。该帧是波形反转后再扩展阶段的峰值帧，也是整个周期的中点。

## 功能点目的

1. **周期中点**：36 帧循环的正中间（18/36）
2. **视觉峰值**：展示波形反转后的最高条形
3. **对称中心**：与第 1 帧形成周期对称

## 具体技术实现

### 文件规格
- **尺寸**：17 行 x 40 列
- **字符集**：Unicode 方块元素字符
- **文件大小**：1222 字节（较大，内容丰富）
- **帧索引**：17（0-based）

### 周期中点特征
```
帧 1-17: 前半周期（扩展-收缩-反转-再扩展）
帧 18: 中点/峰值（本帧）
帧 19-36: 后半周期（镜像前半周期）
```

### 技术集成
```rust
// 中点帧索引
const MID_FRAME_IDX: usize = 17; // frame_18

// 帧选择
let idx = ((elapsed_ms / 80) % 36) as usize;
// idx = 17 时返回 frame_18.txt
```

## 风险、边界与改进建议

### 风险
1. **中点突兀**：峰值帧与相邻帧差异过大
2. **对称性破坏**：后半周期与前半周期不对称

### 边界
- 中点索引计算
- 周期循环处理

### 改进建议
1. **对称验证**：自动化验证前后半周期对称性
2. **峰值优化**：优化峰值帧的视觉冲击力
3. **周期分析**：分析完整周期的视觉流畅度
