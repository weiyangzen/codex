# 研究文档: codex_tui__chatwidget__tests__chat_small_idle_h3.snap

## 场景与职责

本快照文件验证 **小尺寸聊天窗口（高度=3）在空闲状态** 的渲染输出。

测试最小可用窗口尺寸下的渲染行为。

## 功能点目的

1. **最小可用尺寸**: 验证高度为 3 的基本可用性
2. **元素分配**: 观察 3 行空间如何分配给各元素
3. **基础 UI**: 验证最小空间下的基本 UI 展示
