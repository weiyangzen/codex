# 研究文档: codex_tui__chatwidget__tests__chat_small_running_h2.snap

## 场景与职责

本快照文件验证 **小尺寸聊天窗口（高度=2）在运行状态** 的渲染输出。

测试运行状态下稍大窗口的渲染行为。

## 功能点目的

1. **渐进测试**: 高度 2 的运行状态渲染
2. **空间分配**: 观察运行状态如何分配有限空间
3. **状态可见性**: 验证运行状态是否可见

## 具体技术实现

### 快照内容
```
"                                        "
"                                        "
```

### 测试参数
- **窗口高度**: 2 行
- **窗口宽度**: 40 字符
- **状态**: Running（运行中）

### 观察
与空闲状态相同，两行均为空白。
说明在高度为 2 时，运行状态仍然无法显示。
