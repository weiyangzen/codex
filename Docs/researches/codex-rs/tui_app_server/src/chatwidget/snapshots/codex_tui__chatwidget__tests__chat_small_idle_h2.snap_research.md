# 研究文档: codex_tui__chatwidget__tests__chat_small_idle_h2.snap

## 场景与职责

本快照文件验证 **小尺寸聊天窗口（高度=2）在空闲状态** 的渲染输出。

测试比 h1 稍大的窗口尺寸下的渲染行为。

## 功能点目的

1. **渐进尺寸测试**: 验证高度为 2 的窗口渲染
2. **布局分配**: 观察有限空间如何分配给各 UI 元素
3. **空白处理**: 验证空闲状态的空白行渲染
