# 研究文档: codex_tui__chatwidget__tests__chat_small_running_h3.snap

## 场景与职责

本快照文件验证 **小尺寸聊天窗口（高度=3）在运行状态** 的渲染输出。

测试运行状态下最小可用窗口的渲染行为。

## 功能点目的

1. **最小运行显示**: 验证高度 3 时运行状态的显示
2. **可用性边界**: 确定最小可用高度
3. **状态优先级**: 验证状态信息的显示优先级

## 具体技术实现

### 快照内容
```
"                                        "
"                                        "
"                                        "
```

### 测试参数
- **窗口高度**: 3 行
- **窗口宽度**: 40 字符
- **状态**: Running（运行中）

### 观察
即使在运行状态，高度为 3 时仍然显示空白。
这可能表明：
- 状态显示需要更多垂直空间
- 测试场景可能未触发实际的状态渲染
- 或者状态渲染被其他元素覆盖
