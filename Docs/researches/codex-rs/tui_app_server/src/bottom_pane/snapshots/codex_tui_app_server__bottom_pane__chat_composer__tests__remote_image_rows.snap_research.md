# 快照分析：remote_image_rows

## 场景与职责

该快照展示了 ChatComposer 处理远程图片时的特殊渲染方式。远程图片以独立行的形式显示在文本区域上方，与粘贴图片的占位符显示方式不同。

## 功能点目的

- **远程图片独立显示**：`[Image #1]` 和 `[Image #2]` 作为独立行显示在 "describe these" 文本上方
- **视觉分离**：将图片与文本输入区域在视觉上分离
- **多图片管理**：支持多张远程图片同时显示

## 具体技术实现

- 远程图片以行（row）的形式渲染在 textarea 上方
- 每张图片占据一行，显示 `[Image #{index}]` 标识
- 文本输入区域位于图片行下方
- 图片行与文本区域保持适当的间距

## 关键代码路径与文件引用

- `codex-rs/tui_app_server/src/bottom_pane/chat_composer.rs` - ChatComposer 组件实现
- 远程图片行渲染逻辑
- 布局管理（图片行 + 文本区域）

## 依赖与外部交互

- 依赖远程图片 URL 列表
- 与图片加载/缓存系统交互

## 风险、边界与改进建议

- **边界**：
  - 大量远程图片时可能占用过多垂直空间
  - 图片加载失败时需要优雅降级
- **改进建议**：
  - 支持图片行折叠/展开
  - 显示图片缩略图或加载状态
  - 支持拖拽调整图片顺序
