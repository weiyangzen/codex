# 快照分析：mention_popup_type_prefixes

## 场景与职责

该快照展示了 ChatComposer 的提及（mention）弹出框功能，当用户输入 `$` 符号和搜索前缀时，显示匹配的插件/技能/连接器建议列表。

## 功能点目的

- **智能提示**：用户输入 `$goog` 时，显示 Google Calendar 相关的插件/技能/应用建议
- **前缀匹配**：支持基于输入前缀的模糊匹配或前缀匹配
- **快捷调用**：允许用户通过 `$` 语法快速调用外部功能

## 具体技术实现

- 监听 `$` 字符触发提及模式
- 捕获 `$` 后的输入作为搜索前缀
- 根据前缀过滤可用的插件/技能/连接器列表
- 弹出框显示匹配项，支持键盘导航和选择

## 关键代码路径与文件引用

- `codex-rs/tui_app_server/src/bottom_pane/chat_composer.rs` - ChatComposer 组件实现
- 提及弹出框渲染逻辑
- 插件/技能/连接器数据源

## 依赖与外部交互

- 依赖插件/技能/连接器注册表
- 与自动完成/建议系统交互

## 风险、边界与改进建议

- **边界**：
  - 匹配算法需要平衡精确度和召回率
  - 大量可用项时弹出框可能过长
- **改进建议**：
  - 支持模糊匹配和拼音搜索（中文场景）
  - 添加最近使用或常用项优先排序
  - 显示插件图标或描述信息增强可识别性
